## 기획서 종류
- UI
- 시스템
- 콘텐츠(레벨디자인, 클래스, 스킬 등)
- 벨런스
  
  
- 분석 기획서  
-- 여러 게임의 특정 파트만 모아서 분석
  
## 기획시 주의사항  
- 기획시 BM과 지표는 필수다.  
- 개선의 방향을 잡고 더 나은 제품을 제시  
- 자기소개는 PPT로 만들면서 시각화  
- 나의경쟁력, 성격, 경력 기술  
- 80페이지도 3분으로 설명    
-- 시스템기획인 경우 의도, 주요정책, 기능 요약, 프로세스 등으로 압축  
-- 타 게임과 비슷한 부분은 언급만 , 차별성에 중점을 두자.
